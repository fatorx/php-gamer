# PHP-Gamer

Instructions for run this app:

First time

```bash
$ git clone git@github.com:fatorx/php-gamer.git
$ cd php-gamer
$ chmod +x docker-build.sh
$ chmod +x docker-ws.sh
$ ./docker-build.sh
$ ./docker-ws.sh
```

Next times

```bash
$ cd php-gamer
$ ./docker-ws.sh
```

- Access in a browser: http://localhost:8007/client.html
